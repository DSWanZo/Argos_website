Argos

Digital Image Correlation Software

Features

  • Digital Image Correlation (DIC) for experimental mechanics
  • Full-field displacement and strain measurements
  • GPU-parallel correlation (CUDA)
  • Heaviside enrichment for crack and discontinuity analysis
  • Customizable derivative and smoothing kernels
  • TIFF export

Microscopy Module Add-on

Optional module for LSCM/SEM imaging applications.

  • Mosaic mode: multi-tile stitching for large specimens
  • Twinning mode: transition thickness optimization for twins and slip bands
  • 3D height maps: out-of-plane displacement from LSCM topography

System requirements

  • Windows 10/11 (64-bit)
  • NVIDIA GPU (minimum: Turing generation, e.g. GTX 16xx / RTX 20xx)

Installation

  1. Extract the archive to a folder of your choice
  2. Run Argos2D.exe

No system installation required. The application is portable.
